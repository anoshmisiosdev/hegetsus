Our Goal

We wanted to build a game on the Unity engine that incorporated elements of the history of Computer Science. After brainstorming, we decided on making a 2D fighting game with 2 playable characters, with each being capable of movement and 2 attacks.
However, once we quickly realized that we didn't have the time to complete such a project halfway through, we once again had to change our project. Our final game is a trivia game, where two players compete to answer the most questions. We decided to make the game competetive to preserve the original idea and carried over many of the original sprites used.



















Creating the Game:

The title screen was one the simplest parts to create, all we needed to do was use Unity UI system to create a button and scrolling background. Unity AudioSource and other audio features provided the music. The button itself uses the SceneManager and SceneIndex to switch scenes to the Fight Screen.
The main challenge in making this section of the game was in creating a variable that could be accessed by all gameObjects, which we solved by using the static type. Using a control variable and string functions to create a scoreboard, and an array to sort the questions, the two player Quiz game was created. The questions were created by ourselves.
