AR Glasses

Our Projects-CSP Website

Our Goal

The AR Glasses were our attempt at using all our prior knowledge to create an innovative, usable project. They display time onto a glasses lense by utilizing a mirror and an OLED screen. The project itself required us to code an arduino and OLED display, as well as design and build a frame to mount the electronics conveniently.
The glasses use code written in C along with the AdaFruit library to convert text instructions into code the OLED display knows how to process. The frame was designed by accurately measuring the dimensions of the glasses and circuitry and then using those to create a 3D model that we could print and mount things to.
